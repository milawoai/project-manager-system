# 分布式任务分发系统 - 工作计划

## 项目概述

构建分布式任务分发平台，服务端使用 NestJS + MySQL，工作机器使用 Node.js + SQLite。

---

## 阶段一：服务端基础架构

### 1.1 NestJS 项目初始化

**任务列表：**

| 序号 | 任务 | 说明 |
|------|------|------|
| 1.1.1 | 创建 NestJS 项目 | 使用 nest new project-distribute-server |
| 1.1.2 | 配置 TypeORM + MySQL | 连接远程 MySQL 数据库 |
| 1.1.3 | 配置 WebSocket | 安装 @nestjs/websockets + @nestjs/platform-ws |
| 1.1.4 | 配置 Swagger | API 文档生成 |
| 1.1.5 | 配置 Docker | 编写 Dockerfile |

**产出：**
- 可运行的 NestJS 服务端基础项目
- Docker 镜像构建配置
